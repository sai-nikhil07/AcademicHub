# Student Management System

A web-based **Student Management System** built using **Django** to manage academic activities efficiently.  
This project provides role-based access for **Admin, Teachers, and Students** and helps streamline student records, subject management, results, and internal communication.

---

##  Features

- Role-based authentication (Admin, Teacher, Student)
- Student and Teacher management
- Subject and course allocation
- Result and performance tracking
- Internal messaging / communication system
- Admin dashboard for centralized control
- Secure login and authorization

---

##  Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite
- **Version Control:** Git & GitHub
